# 🥬 김치 Phytate 스터디

김치 전문가 과정 과제 — **김치 유산균으로 현미의 피트산(항영양소)을 줄이는** 연구와 특허를
해설·퀴즈·보고서 가이드와 함께 배우는 학습 앱 + 보고서 작성 워크스페이스.

## 🔗 학습 앱 (GitHub Pages)
👉 **https://jwj-nick.github.io/kimchi_study_2606_01/**

- 📄 논문 원문 보기(공식 오픈액세스 링크)
- 📚 챕터 8개 해설 학습 (읽음 체크 → 진행률 저장)
- ✏️ 이해도 퀴즈 초급·중급 (모든 챕터 읽으면 잠금 해제)
- 💭 생각해볼 주제 · 🔗 연관 주제 탐색(검색 도우미)
- 📝 보고서 작성 가이드 + 현재 보고서

## 📚 다루는 자료
- **논문:** 김치로부터 Phytate 분해 유산균 선별 및 현미에서 반응특성 — 박성희 외, 한국식품영양과학회지 42(4):627–632 (2013). [DOI](https://doi.org/10.3746/jkfn.2013.42.4.627)
- **특허:** *L. sakei* Wikim001 발효현미 제조방법 — [KR101465547B1](https://patents.google.com/patent/KR101465547B1/ko), 한국식품연구원 (2014)
