# kimchi-start — 현황 점검 & 다음 할 일

비개발자 수강생을 돕는 세션이다. 항상 **쉬운 한국어, 한 번에 한 단계씩**.

## 할 일
1. `PLAN.md`의 체크리스트(2번 항목)를 읽어 현재 어디까지 됐는지 파악한다.
2. `papers/`, `study/`, `report/`, `app/` 폴더 내용을 확인한다.
3. 아래 형식으로 짧게 보고한다:
   - **지금까지:** (완료된 것 1~3줄)
   - **지금 단계:** (한 줄)
   - **다음 한 가지:** (구체적 행동 1개 + 부를 명령 1개)
4. 사용자가 막혔으면 선택지를 2~3개로 좁혀 제시한다. 절대 한꺼번에 여러 단계를 시키지 않는다.

## 규칙
- 전문용어는 풀어서 설명. 다음 행동은 항상 한 줄로.
- 사실 확인이 필요하면 웹 검색하고 출처를 남긴다.
