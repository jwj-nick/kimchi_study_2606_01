# kimchi-study — 논문 읽고 스터디 노트 만들기

## 입력
대상 PDF 경로(`papers/...`). 없으면 `papers/`에서 고르게 한다.

## 절차
1. Read 도구로 PDF 전체를 읽는다(여러 쪽이면 나눠 읽기).
2. 아래 틀로 `study/NN_제목.md`를 작성한다(NN = 순번):
   - **기본 정보(출처):** 제목·저자·소속·학술지/권호·쪽·연도·DOI·파일명
   - **한 문장 요약**
   - **왜 했나(문제)**
   - **어떻게 했나(방법, 쉽게)**
   - **핵심 결과(숫자)** — 보고서에 쓸 수치 강조
   - **한 줄 결론**
   - **한계** — 솔직히
   - **보고서에 쓸 핵심 숫자 후보** (1개, 출처 함께)
   - **짝꿍 특허 아이디어** (다음 단계 검색어)
3. 끝에 "다음 한 가지"를 안내.

## 규칙 (저작권)
- 본문 그대로 복사 금지. 항상 자기 말로 요약.
- 직접 인용은 15단어 미만·출처당 1회·따옴표+출처.
- 표/그림은 수치만 골라 다시 정리.
- 논문에 없는 사실을 지어내지 않는다. 모르면 "논문에 없음"으로 둔다.
- 전문용어(종균·내산성·nisin 등)는 괄호로 쉬운 풀이를 단다.
