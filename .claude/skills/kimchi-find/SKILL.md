# kimchi-find — 논문/특허 찾기 & 다운로드

## 입력
사용자가 준 주제·키워드. 없으면 `topics.md`와 `PLAN.md`의 확정 주제를 참고해 키워드를 제안한다.

## 절차
1. **검색:** WebSearch로 후보를 찾는다. 논문은 학회지·DOI, 특허는 Google Patents/KIPRIS 우선.
2. **후보 정리:** 3개 이내로 추려 각각 한 줄 요약 + 출처(학술지/특허번호·연도) + 무료여부를 표로 제시.
3. **다운로드 판단:**
   - **오픈액세스(e-jkfn.org, koreascience.kr 등):** PDF 직링크를 찾아 `curl -sL -A "Mozilla/5.0" -o papers/<알기쉬운이름>.pdf <url>`로 받는다. 받은 뒤 `file`로 PDF인지, Read로 1쪽 내용이 맞는지 **반드시 검증**.
   - **로그인 필요(사이언스온·DBpia):** 직접 못 받음. 검색 키워드 + 찾는 법을 단계로 안내하고, 사용자가 `papers/`에 저장하게 한다.
   - **특허:** Google Patents 화면을 그대로 봐도 됨. 특허번호·출원인·핵심 청구를 요약.
4. **파일명 규칙:** `papers/주제_연도_출처.pdf` (예: `김치저장성_항균유산균_2014_JKFN.pdf`).
5. 끝에 "다음 한 가지"를 한 줄로 안내(보통 `kimchi-study`).

## 규칙
- 무료 PDF만 자동 다운로드. 유료/로그인 콘텐츠는 우회 다운로드 시도 금지.
- 다운로드 후 실제 논문이 맞는지 1쪽을 열어 제목·저자를 확인하고 사용자에게 보고.
- 불확실한 서지정보는 원문으로 확인. 최종 사실 확인은 사이언스온 원문 기준.
