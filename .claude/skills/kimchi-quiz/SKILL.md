# kimchi-quiz — 스터디 확인 퀴즈 앱

## 목적
수강생이 논문 내용을 제대로 이해했는지 스스로 점검하게 한다. 빌드 없이 더블클릭으로 열리는 단일 HTML.

## 절차
1. `study/`의 노트들을 읽어 문제 재료(핵심 숫자·결론·용어·한계)를 뽑는다.
2. 문제를 JSON 배열로 만든다. 각 문항:
   - `q`(질문), `choices`(4지선다) 또는 `type:"ox"`, `answer`(정답 인덱스/불리언), `why`(해설 = 출처·핵심 한 줄)
   - 난이도 섞기: 사실확인(숫자) / 개념(왜) / 한계(비판적). 노트 1편당 5~8문항.
3. `app/kimchi_quiz.html` 단일 파일로 작성한다:
   - 바닐라 HTML/CSS/JS, 외부 의존성 없음(CDN도 가급적 X), 한국어 UI.
   - 기능: 문항 순차 풀이, 즉시 정오 + 해설 표시, 진행률, 최종 점수, 다시풀기, 오답만 다시.
   - 문제 데이터는 파일 상단 `const QUESTIONS = [...]`로 분리해 나중에 손쉽게 추가.
4. 다 만들면 사용자가 여는 법을 한 줄로: "파일 탐색기에서 `app/kimchi_quiz.html` 더블클릭".

## 규칙
- 정답·해설은 반드시 스터디 노트/논문 근거에 일치. 지어내지 않는다.
- 새 논문 노트가 생기면 기존 QUESTIONS에 덧붙여 갱신.
- 디자인은 단순·읽기 쉽게(큰 글씨, 모바일에서도 OK).
